## Getting Started
Follow these instructions to run the project locally.

### App Installation

1. **Clone the repository**
```bash
git clone https://github.com/sobselena/ozon-test-task.git
```
2. Go to the project folder
  ```bash
cd ozon-test-task
```
3. **Install dependencies**
```bash
npm install
```

4. **Start the app**
```bash
npm run dev
```
